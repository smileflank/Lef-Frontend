# data-lef (Lef Form/Data Management API)
data-lef is light html form and sql data management JS API. It has not been fin-
ished yet. I will be really helpful for web developers.
data-lef 是一款轻量级的：HTML表单及数据库管理JS插件，目的是将大量重复的表单验证、
数据库后台管理的功能解耦成独立的JS插件。data-lef 是根据我以前实际作业经验而总结
的一套方案，暂时还未完成，但是相信完成后，其作用一定是非常广泛的。
